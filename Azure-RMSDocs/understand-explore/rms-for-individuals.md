# <a name="rms-for-individuals-and-azure-information-protection"></a>RMS para individuos y Azure Information Protection

>*Se aplica a: Azure Information Protection*

RMS para individuos es una suscripción gratuita de autoservicio para los usuarios de una organización que necesitan abrir archivos que han sido protegidos por el servicio Azure Rights Management de Azure Information Protection. Si estos usuarios no se puede autenticar con Azure Active Directory y su organización no tiene Active Directory Rights Management (AD RMS), este servicio de suscripción gratuita puede crear una cuenta en Azure Active Directory para un usuario. El resultado es que estos usuarios podrán autenticarse con su dirección de correo de empresa y abrir los archivos protegidos en sus equipos o dispositivos móviles.

Al usar el [cliente de Azure Information Protection](../rms-client/client-user-guide.md) en equipos Windows, estos usuarios también pueden proteger sus propios archivos, pero esta capacidad está pensada solo para usarla en versiones de prueba y podría eliminarse en el futuro.

RMS para usuarios es un ejemplo de una suscripción de autoservicio que es compatible con Azure Active Directory. Para más información sobre cómo funciona, consulte [¿Qué es la suscripción de autoservicio de Azure?](/active-directory/active-directory-self-service-signup) en la documentación de Azure Active Directory. 

> [!IMPORTANT]
> Esta suscripción gratuita constituye una opción para ayudar a garantizar que las personas autorizadas ajenas a la organización siempre puedan leer archivos protegidos por la organización. Otra opción sería enviar por correo electrónico los documentos siguiendo los pasos de [Cifrado de mensajes de Office 365 con nuevas capacidades](https://support.office.com/article/7ff0c040-b25c-4378-9904-b1b50210d00e). Esta solución de correo electrónico funciona con todas las direcciones de correo en todos los dispositivos y es la manera recomendada para compartir de forma segura información y documentos de Office por correo electrónico con personas ajenas a la organización.
